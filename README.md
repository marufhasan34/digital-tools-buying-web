Project Name :
Digital Tools Buying Web App

Description :
This is a modern and responsive web application where users can explore and purchase various digital tools. The website provides a clean UI, smooth user experience, and organized product display to help users easily browse tools and manage their selections.


Technologies Used :
⚛️ React.js
🎨 Tailwind CSS
🌼 DaisyUI
⚡ Vite
📦 JavaScript (ES6+)



Features :
🛒 Add to Cart Functionality – Users can select and manage tools easily.
🎯 Dynamic Tool Display – Data-driven UI using JSON and API-based rendering.
📱 Responsive Design – Fully optimized for mobile, tablet, and desktop devices.
